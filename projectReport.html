Crime rate peaks during lunch hour and rush hours in San Francisco

Problem discription

The goal of this study is to visualize the data of crimes in San Francisco area, and show that the crime rate reaches peak during lunch hour and rush hoursin San Francisco. The pattern of the crime is relatively consistent across the summer months (June, July and Auguest) in 2014.

R code for visualization

R code is used to visualize and analyze the data. ggplot2 package and reshape is required. Below is the R code for visualization. g1, g2, g3 are the objects of the graphs.

library(reshape)
library(ggplot2)
sf_inc <- read.csv('sanfrancisco_incidents_summer_2014.csv', 
                   stringsAsFactor = FALSE);
HourMin <- strsplit(sf_inc$Time, split=":");
Date <- strsplit(sf_inc$Date, split="/");
dataSz <- dim(sf_inc)[1];
hours <- rep(NA, dataSz);
month <- rep(NA, dataSz); 

for (k in 1:dataSz) {
    hours[k] <- HourMin[[k]][1]
    month[k] <- Date[[k]][1]

}
sf_inc$hours <- hours;
sf_inc$month <- month; 
#weekday_inc <- sf_inc[sf_inc$DayOfWeek != 'Sunday' & sf_inc$DayOfWeek != 'Saturday',]
#weekday_hours <- weekday_inc$hours
#weekend_inc <- sf_inc[sf_inc$DayOfWeek == 'Sunday' | sf_inc$DayOfWeek == 'Saturday',]
#weekend_hours <- weekend_inc$hours

#hist(hours, breaks = 24)
#peak hours 12, 17, 18
peakHour12 <- sf_inc[sf_inc$hours == '12', ]$Category
peakHour1718 <- sf_inc[sf_inc$hours == '18' | sf_inc$hours == '17', ]$Category
patternFreq_12pm = as.data.frame(table(peakHour12))
patternFreq_1718pm = as.data.frame(table(peakHour1718))
colnames(patternFreq_12pm) <- c("pattern", "Freq_12pm")
patternFreq_12pm$Freq_12pm <- 2*patternFreq_12pm$Freq_12pm
colnames(patternFreq_1718pm) <- c("pattern", "Freq_1718pm")
patternFreq <- merge(patternFreq_12pm, patternFreq_1718pm)
patternFreq.m = melt(patternFreq)
colnames(patternFreq.m)[3] <- "Freq"
g1<-ggplot(patternFreq.m, aes(pattern, Freq)) +  geom_bar(aes(fill = variable), position = "dodge", stat="identity") + coord_flip()
g1 <- g1+ ggtitle(expression(atop("comparison of lunch hour crime and rush hour crime")))
 

peakHour12_June <- sf_inc[sf_inc$hours == '12' & sf_inc$month == '06', ]$Category
peakHour1718_June <- sf_inc[(sf_inc$hours == '18' | sf_inc$hours == '17') & sf_inc$month == '06', ]$Category

peakHour12_July <- sf_inc[sf_inc$hours == '12' & sf_inc$month == '07', ]$Category
peakHour1718_July <- sf_inc[(sf_inc$hours == '18' | sf_inc$hours == '17') & sf_inc$month == '07', ]$Category

peakHour12_Aug <- sf_inc[sf_inc$hours == '12' & sf_inc$month == '08', ]$Category
peakHour1718_Aug <- sf_inc[(sf_inc$hours == '18' | sf_inc$hours == '17') & sf_inc$month == '08', ]$Category

patternFreq_12_June = as.data.frame(table(peakHour12_June))
colnames(patternFreq_12_June) <- c("pattern", "Freq_June_12pm")
patternFreq_12_July = as.data.frame(table(peakHour12_July))
colnames(patternFreq_12_July) <- c("pattern", "Freq_July_12pm")
patternFreq_12_Aug = as.data.frame(table(peakHour12_Aug))
colnames(patternFreq_12_Aug) <- c("pattern", "Freq_Aug_12pm")
patternFreq_12 <- merge(patternFreq_12_June, patternFreq_12_July)
patternFreq_12 <- merge(patternFreq_12, patternFreq_12_Aug)

patternFreq_1718_June = as.data.frame(table(peakHour1718_June))
colnames(patternFreq_1718_June) <- c("pattern", "Freq_June_1718pm")
patternFreq_1718_July = as.data.frame(table(peakHour1718_July))
colnames(patternFreq_1718_July) <- c("pattern", "Freq_July_1718pm")
patternFreq_1718_Aug = as.data.frame(table(peakHour1718_Aug))
colnames(patternFreq_1718_Aug) <- c("pattern", "Freq_Aug_1718pm")
patternFreq_1718 <- merge(patternFreq_1718_June, patternFreq_1718_July)
patternFreq_1718 <- merge(patternFreq_1718, patternFreq_1718_Aug)

patternFreq_12.m = melt(patternFreq_12)
colnames(patternFreq_12.m)[3] <- "Freq"
g2<-ggplot(patternFreq_12.m, aes(pattern, Freq)) +  geom_bar(aes(fill = variable), 
           position = "dodge", stat="identity") + coord_flip() + ggtitle(expression(atop("frequency distribution of crime patterns during lunch hour")))

patternFreq_1718.m = melt(patternFreq_1718)
colnames(patternFreq_1718.m)[3] <- "Freq"
g3<-ggplot(patternFreq_1718.m, aes(pattern, Freq)) +  geom_bar(aes(fill = variable), 
         position = "dodge", stat="identity") + coord_flip() + ggtitle(expression(atop("frequency distribution of crime patterns during rush hour")))

Supporting evidence from the data

The first supporting evidence is that, in general, the crime rates peaks around 12:00pm, which is the lunch hour; and around 17:00pm and 1 hour after that, which is the rush hour.

ggplot(sf_inc, aes(x=hours)) + geom_bar()

More detailed data shows that the crime patterns during lunch hour and rush hour have subtle differences. Note that, since lunch hour is just 1 hour, and rush hour is 2 hours, to make the comparison fair, the frequency number in rush hour is timed by 2.
In general, larcenry/theft is the most common, and it shows a noticeable increase during rush hours.
Vehicle theft also increases during rush hour, and missing person is more likely to happen during lunch hour. In the plot below, Freq_12pm stands for frequency of the crime pattern during lunch hour around 12:00pm, and Freq_1718pm stands for frequency of the crime pattern during rush hour around 17:00pm and 18:00pm.

To show more details, the data from June to Auguest are visualized. It appears that the crime patterns also keep relatively consistent across different months in summer (June, July, Augest). However, noticeble increase in larcenry/theft is observed from June to Auguest during both lunch hour and rush hours. Summary

The crime rate peaks during lunch hours and rush hours in San Francisco area, and the crime pattern is consistent in this two time slots across the three summer months.
